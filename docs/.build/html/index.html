Installation¶

Now installation is being simple, you can just use pip to install:

pip install ancypatch

If you still have some problem install dependencies, you can still run deps.sh, which is legacy.(I may change that when it is appropriate) And then run

python setup.py install

Note that this setup.py script will not install dependencies for you.

hook(src, dst, first=False, noentry=False)[source]¶

hooks some address to redirect to user-defined address

Note that if overwritten instructions(when insert call to instructions of yourself) contains jump, it may break.

This is caused by its implementation. When we run the original code hooked after inserting our own code, some instructions after original code are not available. If the original code contains jump to these address, then we break. Try to avoid this by using it another way.

  • src (int) – hooking address, you can also use Context.entry to hook the entry point
